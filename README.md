# PortfolioPro

Portfolio immersif inspiré de l’univers GTO, conçu pour présenter mon profil, mon parcours et mes points de contact à travers une interface cinématique et interactive.

## 🎯 Concept

Ce projet propose une expérience utilisateur scénarisée, avec une direction artistique forte basée sur le personnage d’Eikichi Onizuka.

Objectif : se démarquer des portfolios classiques en combinant développement front-end avancé et identité visuelle marquée.

---

## 🚀 Aperçu

* Navigation scénarisée avec transitions fluides entre les sections
* Expérience immersive (vidéo de fond, overlays dynamiques)
* Interface inspirée des menus RPG
* Navigation clavier + souris
* Design responsive (desktop & mobile)

---

## 🧠 Features techniques

* Gestion des transitions complexes avec Framer Motion
* Routing dynamique avec React Router
* Optimisation des performances (lazy loading, découpage des composants)
* Gestion des animations synchronisées avec l’UI
* Architecture modulaire React
* Site web 100% responsive

---

## 📂 Sections

* Home : écran d’entrée + menu principal
* About : présentation, vision, objectifs
* Resume : compétences, formation, projets
* Socials : GitHub, LinkedIn, contact

---

## 🛠️ Stack

* React 19
* Vite
* React Router
* Framer Motion
* CSS
